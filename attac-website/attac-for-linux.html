WOW!! A LINUX helper? Are you pulling my leg cause i am really excited.
We are not pulling your leg. ATTAC v4 is compiled for LINUX also. We are compiling it for the Intel processor

What other systems are you compiling it for?
We will probably compile ATTAC for Free BSD because Mac OS X might be able to run it if it is compiled for Free BSD

Is ATTAC for LINUX going to be faster than any other helper?
Probably, since LINUX is just faster in general, ATTAC v4 for LINUX will probably be the fastest helper around.

Do i need to pay for ATTAC for LINUX seperately? Do we need a new reg code?
Nope. Purchasing ATTAC once gets you the windows version and the LINUX version. The codes for the windows version and the LINUX version are the same.

Is the LINUX version different than the windows version?
Not at all. They are exactly the same. Since we use the same source base to compile both the windows and LINUX version, the LINUX version will never lag behind in bug fixes since anything fixed in the windows version is fixed in the LINUX version.

What was it you said before about ATTAC working on Macs?
To our understanding, Mac OS X is based on Free BSD code. Therefore, we think that if we compile ATTAC v4 for Free BSD (which we can) then it should run on Mac OS X.

Why dont any of the functions like Colonize work?
As this is only a beta version, we havent plugged that in yet. We will be adding it shortly. For now, you can use the REXX engine to run your scripts on ATTAC v4 as it is fully functional.
