(7/16/2001 19:30:00) ATTAC version 4 BETA 2 released

This fixes some of the registration problems as well as the async problem and the freezing problem. Small changes to the rexx engine also. Some other bug fixes included. Read the changelog.
For BETA 2, you need to just reenter your codes. You can install right over the old version. Your game files will not be affected. First delete the reg.txt file and then start v4 BETA 2 and reenter the same codes you entered for v4 BETA 1.

(7/10/2001 23:00:00) ATTAC version 4 BETA 1 released

ATTAC releases version 4 BETA 1 and the first tradewars helper for LINUX. ATTAC v4 is compatible for Windows 95/98/NT/2000 and for LINUX. ATTAC v4 for LINUX is compiled for Intel processors. This is a BETA version so not everything is functioning. Things will be added over the next few weeks and we look to release a full version shortly. 
Please note that a debug window will show start up with ATTAC v4. This will be removed in the full release version. To speed up ATTAC v4, minimize this window.

Reg codes will be sent to registered users over the next week. The same reg code for windows will work on the LINUX version.

ATTAC v3.1.1 is the last stable version and will still be available for download on the download page.

(7/10/2001 23:00:00) New Website design

We created a new look for the ATTAC website to make it feel more like a Tradewars website. Hope you like it. Reorganized the menus a little.
