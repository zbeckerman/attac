What exactly am I buying?
ATTAC With TWXSync Integration: This purchases 1 registered version of ATTAC and will allow you to connect to a registered version of TWXSync server to do realtime data synchronization. The intergration ability ALLOWS you to connect as a client to a TWXSync server. The server is a seperate piece of software that is purchased seperately.

Purchase ATTAC Without TWXSync Integration: This purchases 1 registered version of ATTAC and will NOT connect to a TWXSync server to do realtime data synchronization. Even if the TWXSync server is registered, the client must be properly enabled via registration.

Upgrade previously registered version of ATTAC to include TWXSync Integration: Anyone with a previous ATTAC registration code can upgrade their registration to allow them to connect to a TWXSync server. This is for registered ATTAC users only. The intergration ability ALLOWS you to connect as a client to a TWXSync server. The server is a seperate piece of software that is purchased seperately.
